# Writing an interpreter and a compiler

There are 2 languages implemented in this repo based on different books.
- monkey -> `monkey.md`
- lox -> `lox.md`

Go to the respective folder for running the code.

## Quickstart

```
cd lox
cargo test

# or
cd monkey
cargo test
```
